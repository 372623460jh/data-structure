# 指令

```
# 删除 node_modules,和 lib 的构建产物，并重新安装，构建
npm run init
# 删除 node_modules,和 lib 的构建产物
npm run clean
# 构建
npm run build
# watch: 构建(监听)
npm run build
# 构建(生产环境)
npm run build:production
# 启动 lib 调试服务
npm run start
# 推包到 npm
npm run publish
# lint&&test
npm run ci
```
